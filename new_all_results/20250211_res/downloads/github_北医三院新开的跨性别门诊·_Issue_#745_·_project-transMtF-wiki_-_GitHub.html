    医院名称

    北京大学附属第三医院

    医生姓名

    常旭

    标签

    提供 HRT 指导, 需家长同意, 有检查项目

    可开具的药品

    戊酸雌二醇, 雌二醇水合物, 醋酸环丙孕酮, 螺内酯

    注意事项

    常旭医生在北医三院美容成形开了专门的跨性别门诊。时间是周一上午成形科（和潘柏林同一个入口）。

    另外常旭在周五下午也会在成形科的普通门诊。普通门诊没有指定医生，所以需要在护士站分诊台指定需要看常旭医生。

    有可以提供醋酸环丙孕酮，但是需要好大夫网站上线上问诊之后邮寄到家。

    门诊说明

    为帮助大家了解就诊途径、保证门诊质量，现将北医三院跨性别医疗门诊时间、挂号途径、就诊须知说明如下：

    跨性别相关内容如跨性别初诊咨询、启动内分泌治疗、激素治疗复诊、性别重置主体手术咨询等服务，均需挂“易性症门诊”就诊，其他门诊时段不接诊上述医疗咨询和复诊。

    目前团队共有三位医生开设易性症门诊，以上项目可在任意医生处就诊。且因目前就诊量大，为保证每位来诊者的就医质量，均需预约挂号。无特殊情况，不予现场加号。

    潘柏林医生、常旭医生的其他门诊时间，仅接诊美容整形的来诊者，其中可包括跨性别整形手术（非性别重置主体手术）的咨询。

    感谢大家对北医三院跨性别医疗团队的信任和理解。
