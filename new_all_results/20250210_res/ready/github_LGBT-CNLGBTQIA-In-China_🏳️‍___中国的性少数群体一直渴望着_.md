# LGBTQIA in China

🏳️‍🌈 中国的性少数群体一直渴望着自由平等

本仓库意图在 LGBTQIA 程序员群体中唤起对自由平等的追求，哪怕我们都深知这不会激起浪花。

你可以访问[这个网站](https://lgbt-cn.github.io)查看更多信息。
