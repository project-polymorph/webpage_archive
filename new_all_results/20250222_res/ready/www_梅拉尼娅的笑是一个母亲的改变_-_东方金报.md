# 梅拉尼娅的笑：一个母亲的改变

作者 : 惠媛
时间：2024-11-11 10:16:13
来源：东方金报网
围观 : 6 次 评论 : 0

本周最火的就是川普再次当先美国总统了。一般的总统当两届是一任，但是建国同志不一样，两届必须是两任。是继克利夫兰之后的第二人，是20世纪以来的第一人，也是共和党第一人。不愧是建国啊。‍‍‍‍‍‍‍‍‍‍‍‍‍

最被网友热议的是第一夫人梅拉尼娅，这次她真的是全力以赴为丈夫拉选票，建国宣布获胜的时候得意洋洋、发自肺腑地感谢梅拉尼娅的时候，咱梅姐的笑容灿烂极了。‍‍‍‍‍‍‍‍‍‍‍‍

![](/d/file/2024/11/11/rzxxx2oib0k.jpg)

和上一次当选以及履职期间梅姐总是冷脸不同，这次梅姐不仅笑的灿烂，连衣服也从明艳多彩的高定变成了行政套装。有媒体认称：梅姐现在改走权力风了。超模就是超模，什么风格都能驾驭。虽然马上又要从豪宅搬到老破小了，但是梅姐这次可以说是心甘情愿。‍‍‍‍‍‍‍‍‍‍‍

![](/d/file/2024/11/11/55mvynrirre.jpg)

有考证认为梅姐的变化是因川宝遭遇枪击改变的。我认为这只是一个契机。更重要的是川宝本次竞选的许多竞选策略是梅姐最爱的巴伦献计献策的，而且巴伦现在也是佛州的党代表了，地表最强二代。‍

所以有人说，梅姐才是真的钮祜禄 梅拉尼娅。这次回宫是要为儿子铺路。加上第一次有第一女儿履行了不少第一夫人的职责，梅姐为了避免家庭矛盾就主动让贤。何况第一夫人不能穿高定、不能拿爱马仕，梅姐可真不爱这样的生活。但是在这一任里，梅姐不仅是名副其实的第一夫人，还可能是未来的第一母亲，那必须得认真干，并且是要干就干到最好的那种。‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

不穿高定算什么，不拿爱马仕算什么，权力套装走起来，权柄不需要手袋！‍‍‍‍‍

这就是梅姐的成长，丈夫的事业比不上丈夫的财产，但儿子的前途高于一切。毕竟这种offer 的机会不多，必须认真工作每一秒。这就是熹妃说的“父母之爱子必为之计深远”。能使一个女人发自内心改变自己的，从来不是男人而是孩子。‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

这几天回老家和以前的朋友相聚，发现大家现在活得都挺通透的。但是当下自媒体的过度发达人们也喝了不少毒鸡汤。

![](/d/file/2024/11/11/0cdfeluw1in.jpg)

‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

家中一个年轻的晚辈问我说，没钱就不能生孩子了吗？没钱还响应国家号召多生孩子到底得罪了谁？这个问题真的就很难回答。从政策层面讲，我认为所有适龄的人都应当响应国家号召。但是怎么响应就要看各人的实际情况了。年纪、身体以及经济都是考虑因素。单纯从经济角度来说，我觉得不需要看任何人的脸色，只需要考虑自己能不能对得起将来的那个孩子以及现在可能已经有的孩子就行了。‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

一个普通家庭养两个孩子和养一个孩子是不一样的。除了钱还有时间和精力等。大家可以去看梅姐的采访。当年川宝想让她多生几个，毕竟巴伦可太好看了，但是梅姐拒绝了。她说，一个孩子事无巨细就忙不过来，而川宝又没时间。她太知道多孩家庭和一孩家庭的差别了。‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

另一个年轻的后辈说，她可管不了这么多。反正家里让她生她就生，因为公婆说了她只管生就行，至于带孩子家中有的是人。她也很享受这种生活。她还说，“我首先是我自己，其次才是妈妈。”‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

这几年自媒体宣扬了太多“爱自己”的论调，但是究竟什么才是真正的爱自己，怎么样才是最好的爱自己，应该每个的解读都不同。‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

中国女人一辈子都是先孝父母、再服从丈夫、再全身心地为儿女，从来没有自己，所以中国女人最需要爱自己。但这绝不是说爱自己就可以忘记自己作为女儿、妻子和母亲的责任和义务。‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

女儿、妻子和母亲，如果非要排序的话我认为母亲最重要。因为作为儿女有可能是可以分担的，即使是独生子女有一些也是可以通过经济手段分责的，作为夫妻实在不行是可以离婚的（虽然现在离婚不易），只有母亲这个角色，事关孩子的一生，是唯一不可替代。（注夫妻关系仍应为家庭关系的核心，即是对自己负责也是对孩子负责。之所以母亲这个身份更重要，是因为不可替代。）‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

从我本人角度来说，作为母亲，我首先是我自己的含义是我应该成为一个强大的人，有自己的工作，在任何情况下有独立生活的能力，而这些能力的目标是为了更好地保护孩子、更好地助力孩子的成长，让他/她可以更好地拥有属于自己的人生。‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

如果不能给孩子托举那至少也得兜个底。这些都是我个人的观点，不杠。杠就是你对。‍‍‍‍‍‍

其实谈到爱自己以及爱孩子的问题，我真建议大家去看芒果的综艺。我以前是最不屑看综艺的，因为都是剧本。红的无非是更红，糊的来翻红一把。但是今年这几个综艺，剧本都写不了这么精彩。当然如果你是情绪化的人不建议观看，毕竟谁的乳腺不得珍惜呢。‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

不带一丝情感去看，你会发现不论是周学还是麦学都是值得研究的。把它作为现实版的职场剧进行研究，保有收获的。最不济，也会发现自己的工作搭子都比周强、自己的伴侣都比麦好。幸福感直线飙升。
